Allocates a VS and returns it's ID.

Parameters
gsthe gs_t to use
tlsthe tls_t to use
mut_systhe bf_syscall_t to use
intrinsicthe intrinsic_t to use
vpidthe ID of the VP to assign the newly created VS to
ppidthe ID of the PP to assign the newly created VS to
Returns
Returns ID of the newly allocated vs_t. Returns bsl::safe_u16::failure() on failure.

NOTE:

  • Ask the microkernel to create a VS and return the ID of the newly created VS.

NOTE:

  • Once a VS has been created, the microkernel returns the ID of the newly created VS. We can use this ID to determine which vs_t to allocate.
